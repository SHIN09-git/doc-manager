# 隐私政策草案

本文档是“摹文拟笔工作台”商业化灰度阶段的隐私政策草案，正式上线前应由法务或合规负责人复核。

## 数据模式

工作台支持两种模式：

- 本地模式：文档、执笔人、设置和备份主要保存在用户浏览器 IndexedDB 中。
- 云端模式：用户主动登录后，可把文档、执笔人、API Key 配置和 AI 用量同步到部署方后端。

## 可能收集的数据

云端模式可能保存：

- 账号邮箱、姓名、组织和成员关系。
- 用户主动同步的文档和执笔人。
- 执笔人版本、说明和质量报告。
- 组织 API Key 的加密密文和脱敏提示。
- AI 请求用量、任务类型、成功失败状态和审计日志。
- 用户提交的内测反馈。

## AI 调用说明

当用户使用 AI 起草、段落改写、执笔人生成、PPT 生成等功能时，相关提示词、文档片段、样本摘要或执笔人规则可能被发送到用户配置的 AI 服务或云端 AI 代理。

使用真实材料前，用户应确认 AI 服务符合所在单位的数据规范。

## 数据导出与删除

云端面板提供“导出我的云端数据”和“删除云端账号”入口。删除账号会停用云端账号并清除当前会话；本地浏览器中的数据不会被自动删除。若账号是某个仍有成员组织的唯一所有者，系统会优先把所有者身份交接给该组织管理员；如果没有可交接的管理员，删除会被阻止，需要先处理组织成员或联系管理员，避免组织变成无人可管理状态。

组织级数据删除、备份恢复和审计日志留存策略需要由部署方在正式上线前进一步确认。

## 安全措施

- Session Cookie 默认使用 `HttpOnly` 和 `SameSite=Lax`。
- 生产环境要求强 `APP_ENCRYPTION_SECRET` 和 `SESSION_SECRET`。
- 组织 API Key 使用 AES-GCM 加密存储。
- 云端敏感管理操作要求邮箱验证。
- 成员邀请需要邀请码匹配。

## 联系与反馈

灰度阶段可通过云端面板“内测反馈”提交问题。正式商业化前应补充明确的运营联系邮箱和数据处理负责人。
