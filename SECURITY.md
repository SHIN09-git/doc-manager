# 安全说明

摹文拟笔工作台是本地优先的静态应用。项目本身不内置 API Key，也不提供默认云端存储。

## API Key

- 请不要把真实 API Key 提交到仓库、Issue 或截图中。
- API Key 只应在本机浏览器中配置。
- 共享演示环境时，建议使用临时 Key 或单独项目 Key。

## 文档与样本

- 文档、示范文本、执笔人版本和测试记录默认存储在浏览器本机 IndexedDB。
- 调用 AI 时，用户输入、当前文档片段、示范分析或执笔人规则可能会发送到你配置的 AI 服务。
- 处理含个人信息、合同、财务、医疗、教育档案或内部敏感材料时，请先确认接口提供方和组织合规要求。

## 报告漏洞

请在 Issue 中描述问题类型、复现步骤、影响范围和建议修复方向。不要公开真实 API Key、私密文档或可识别个人信息。
